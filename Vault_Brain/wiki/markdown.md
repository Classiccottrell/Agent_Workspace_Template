## Summary
Markdown is a lightweight markup language that uses plain-text-formatting syntax to create structured documents. Because it is stored as standard text files, Markdown-based notes remain highly portable, long-lasting, and independent of proprietary application databases.

## Key Facts
- Plain text is a format rather than an application-specific feature, ensuring notes remain readable over decades.
- Markdown folders compose cleanly with arbitrary tools, enabling desktop renderers, command-line agents, and backup scripts to work on the same vault without vendor lock-in.
- Using plain markdown files turns a directory into its own database, removing the need for export utilities.
- Durable note systems should pass the test of remaining fully readable and searchable even if the vendor company disappears.
