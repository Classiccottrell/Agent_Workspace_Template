## Summary
MarkItDown is a technology utility that converts various document formats, including PDF, DOCX, PPTX, XLSX, HTML, CSV, ZIP, and images via OCR, into structured Markdown. It is available as a [[claude-code]] plugin that operates locally through a Model Context Protocol (MCP) server, preserving formatting elements such as headings, tables, and lists.

## Key Facts
- Supports document conversion for formats like PDF, DOCX, PPTX, XLSX, images (via OCR), HTML, CSV, and ZIP.
- Delivered as a plugin for [[claude-code]] and distributed through the Passion Factory curated Claude Code Plugins Marketplace.
- The marketplace project repository is hosted at [github.com/pleaseai/claude-code-plugins](https://github.com/pleaseai/claude-code-plugins).
