## Summary
The Claude Platform is the developer portal and API management dashboard for building with Anthropic's Claude models. It provides access to API keys, organization usage and billing settings, prompt workbench, and documentation. The platform hosts the models that power terminal agents such as [[claude-code]] and multi-model projects like [[hermes-agent]].

## Key Facts
- Features three primary models: Opus 4.8 (complex projects, agents, and coding), Sonnet 4.6 (everyday tasks, writing, and cost-efficiency), and Haiku 4.5 (fastest, lowest cost, and high volume).
- Offers an Advisor tool (in Beta) that dynamically consults a stronger model mid-task to increase intelligence while minimizing cost and token usage.
- Includes a Fast mode offering up to 2.5x faster output speeds at premium pricing.
- Provides a Batch API for async workloads, saving 50% on standard API pricing.
- Supports Prompt Caching to reuse prompt prefixes across API calls, reducing input costs by 50–90%.
- Includes management interface for monthly spending limits (e.g., $500 limit) and tracking organization credits.
