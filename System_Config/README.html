Automation Hub

Scheduled background jobs (macOS launchd) that capture clips, maintain the weekly rhythm, and check the system's health.

The jobs

JobWhat it doesWhen
Daily clip ingestFiles new clips into the knowledge wiki as reference pages.daily 07:00 + login
Monday noteCreates the week's note, carries tasks forward, folds in weekend edits.login + Mon 08:00
Friday close-outSummarizes the week into the master index and snapshots the note.Fri 16:30
Health checkProbes every layer and rewrites the status dashboard.login + every 4h
Skill syncSyncs npx-installed skills from ~/.agents/skills/ to ~/.claude/skills/.on install + hourly
friday_archive runs by hand. All jobs have idempotent install_*.sh scripts. Scheduling (auto / manual / skip) is set during ./bootstrap.sh.
